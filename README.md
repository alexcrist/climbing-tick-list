# Climbing Tick List

> My personal rock climbing tick list

https://alexcrist.github.io/climbing-tick-list/

## Development

### Requirements

- `npm (6.0.1)`
- `node (8.11.1)`
- `yarn (1.5.1)`

### Installing dependencies

- `yarn`

### Start the Developement Server

- `yarn start`

### Building

- `yarn run build`

### Publishing

- `yarn run publish`
